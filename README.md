# top-nav

People navigate.  
The system observes, connects, and improves.

top-nav is the canonical Top Navigation Object Architecture.

It bridges familiar human navigation with system intelligence.

Users interact through tabs, groups, labels, routes, and active states while the system maps relationships, tracks movement, detects friction, and learns from activity underneath.
