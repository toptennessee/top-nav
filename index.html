Seeds before the tree.

top-nav is the reusable seed pattern for turning captured input into classified, reusable system meaning.

It begins with capture. Then a functional spine gives the capture a role. Hashtag objects attach measurable meaning. Submit commits the result for later use.

Functional Spine

CAPTURE    raw input enters
STRUCTURE  raw input gets shaped
COORDINATE people, timing, ownership, and movement align
BUILD      ideas become assets or systems
EXECUTE    work gets done
DOCUMENT   decisions and knowledge are preserved
AGGREGATE  scattered inputs become larger patterns
IMPROVE    the system learns from itself

Core Distinction

System Receives
