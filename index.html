top-nav
Classification spine for reusable intelligence

Give the rooms grammar.

top-nav is a universal classification workspace extracted from C10. It turns captured activity into reusable operational intelligence.

The system is not a menu. It is a functional spine: Capture, Structure, Coordinate, Build, Execute, Document, Aggregate, Improve. Each layer tells the workspace what kind of activity is happening.

The interface stays simple. The system receives meaning: what this is, where it belongs, what function it serves, and how it becomes useful again later.
Object

This is not a navbar.

The visible surface mirrors familiar browser behavior, but the purpose is not navigation decoration. The purpose is classification. The workspace lets a user capture something, mark what function it serves, and submit it into a system that can reuse it later.

User Sees

A capture button, smooth popup, spine layers, checkbox tags, and submit.

System Sees

Object type, function, route, state, relationship, and next use.

System Learns

What the workspace is doing repeatedly, where friction appears, and what should improve.

Classification Spine

The eight functions.

These are not tabs in the normal sense. They are the grammar of operational activity. Every captured object can be classified by what it is doing inside the system. 

[ CAPTURE ]    raw input enters
[ STRUCTURE ]  raw input gets shaped
[ COORDINATE ] people, timing, ownership, movement align
[ BUILD ]      ideas become assets or systems
[ EXECUTE ]    work gets done
[ DOCUMENT ]   decisions and knowledge are preserved
[ AGGREGATE ]  scattered inputs become larger patterns
[ IMPROVE ]    the system learns from itself

Object Model

Universal schema.

type ClassificationWorkspace = {
  id: string;
  name: string;
  captureInput: string;
  spine: SpineLayer[];
  selectedLayerId?: string;
  selectedHashtags: HashtagObject[];
  memory: WorkspaceMemory;
};

type SpineLayer = {
  id: string;

  function:
    | "capture"
    | "structure"
    | "coordinate"
    | "build"
    | "execute"
    | "document"
    | "aggregate"
    | "improve";

  purpose: string;
  hashtags: HashtagObject[];
};

type HashtagObject = {
  id: string;
  tag: string;
  meaning: string;

  creates:
    | "classification_value"
    | "routing_value"
    | "relationship_value"
    | "retrieval_value"
    | "learning_value";
};

type ClassifiedObject = {
  id: string;
  sourceText: string;
  functions: string[];
  hashtags: string[];
  status:
    | "captured"
    | "classified"
    | "routed"
    | "preserved"
    | "reused";

  createdAt: string;
};

type WorkspaceMemory = {
  recentClassifications: string[];
  repeatedFunctions: string[];
  underusedLayers: string[];
  improvementSignals: string[];
};
